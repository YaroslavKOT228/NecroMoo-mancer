# 🐮 NecroMoo-mancer

Добро пожаловать в **NecroMoo-mancer** — кликер, созданный специально для **Game Jam**! 

Суть игры проста: кликайте на корову, зарабатывайте очки, покупайте мощные улучшения и станьте самым успешным фермером в истории.

## 🏆 Особенности игры
* **Нестандартный геймплей:** Обычно в кликерах можно кликать бесконечно но в моей игре не так
* **Система прокачки:** Система древа навыков
* **Специально для Game Jam:** разработана в короткие сроки
